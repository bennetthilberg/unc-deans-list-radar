# UNC Dean’s List Radar

UNC Dean’s List Radar is a Chrome extension that adds badges to the Canvas “People” tab showing whether students were on the Fall 2025 Dean’s List.

## Use cases

- Group project partner selection
- Study group formation
- Club recruiting

## Limitations

- The Fall 2025 Dean’s List has 69 duplicate names and 6 triplicate names.
- Name spellings on Canvas may be different from spellings on the Dean’s List.
