# UNC Dean’s List Radar

UNC Dean’s List Radar is a Chrome extension that adds badges to the Canvas “People” tab showing whether students were on the Spring 2026 Dean’s List.

## Install

[Install UNC Dean’s List Radar from the Chrome Web Store](https://chromewebstore.google.com/detail/unc-deans-list-radar/penacohefjelinlplnmekgglnhfcobmc).

## Data source

The extension uses the official [Spring Dean’s List for 2026](https://registrar.unc.edu/spring-deans-list-for-2026/) published by the UNC Office of the University Registrar.

## Use cases

- Group project partner selection
- Study group formation
- Club recruiting

## Limitations

- The Spring 2026 Dean’s List has 59 names that appear twice and 2 names that appear three times.
- Name spellings on Canvas may be different from spellings on the Dean’s List.
